# Assignment-8
## Overview
Assignment-8 is a fully responsive App Store web application built with modern frontend technologies. It features a clean and user-friendly interface where users can browse apps, install them, and manage their installed list. The app demonstrates a real-world app marketplace experience.

## Key Features
- Beautiful and responsive design across all devices  
- App cards displaying essential information  
- Install and uninstall apps with one click  
- Search functionality to quickly find apps  
- Sorting options to organize apps by name, rating, or downloads  
- LocalStorage integration to persist installed apps even after page reload  

## Technology Stack
- **Frontend:** React  
- **Styling:** Tailwind CSS, DaisyUI  
- **Routing:** React Router  
- **State & Storage:** LocalStorage  
- **Other libraries:** React Recharts (for charts/visuals if any)  

## Dependencies
- react  
- react-dom  
- react-router-dom  
- tailwindcss  
- daisyui  
- react-recharts  

## Setup & Installation
```bash
   # Clone the repository
   git clone https://github.com/sifat-arch/Assignment-8.git
   # Navigate to the project folder
   cd Assignment-8
   # Install dependencies
   npm install
   # Run the development server
   npm run dev


```
